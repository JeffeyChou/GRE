# GRE阅读分类

1. 短篇阅读：160字，2-3题，每个section里2-3篇
2. 逻辑单题：80-100字，1题，每个section里1-2篇
3. 长篇阅读：460字，4题，每个section里0-1篇

# 考试形式

屏幕两个区域，左边文章，右边题目
题目一题一题出

# 做题顺序、速度

**答错题不扣分**，考试时不会做不要空着
所有V题的权重一样，成绩只取决于答对题目的数量

10分钟填空：平均每题1分钟
20分钟阅读：平均每题2分钟
